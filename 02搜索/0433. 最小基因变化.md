## 题意

-   从起始字符串，变成目标串的最小变化次数。
-   考查知识点：哈希表、bfs

## 代码

-   思路：将满足条件的字符串存入到队列中，每次变成哈希表中的字符串以后，需要从哈希表中删除这个字符串，因为再变回来肯定路径更长。
-   时间复杂度：$O(3^8N)$

```cpp
class Solution {
public:
    int minMutation(string s, string end, vector<string>& bank) {
        queue<pair<string, int>> q;
        unordered_map<string, bool> mp;
        unordered_set<char> st{'A','C','G','T'};
        for(auto &str: bank) mp[str] = true;
        
        for(int i = 0; i < 8; i++){
            for(auto &c: st){
                if(s[i] != c){
                    char ch = s[i];
                    s[i] = c;
                    if(mp[s]) {
                        q.push({s, 1});
                        mp[s] = false;
                    }
                    s[i] = ch; 
                }
            }
        }
        
        while(q.size()){
            auto [k, v] = q.front();
            if(k == end) return v;
            q.pop();
            for(int i = 0; i < 8; i++){
                for(auto &c: st){
                    if(k[i] != c){
                        char ch = k[i];
                        k[i] = c;
                        if(mp[k]) {
                            q.push({k, v+1});
                            mp[k] = false;
                        }
                        k[i] = ch; 
                    }
                }
             }
        }
        return -1;
    }
};
```
