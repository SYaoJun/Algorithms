## 题目

-   [69. x 的平方根](https://leetcode-cn.com/problems/sqrtx/) E+
-   二分查找

## 思路

-   因为要求的边界是`x<=mid`，所以可以直接套用左模板

```cpp
int bsearch_left(int l, int r){
    while (l < r){
        int mid = l + r + 1 >> 1;
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    return l;
}
```

-   注意：
    -   整数溢出：考虑0和$2^{31}-1$l两种情况

## 代码

-   时间复杂度：$O(logN)$

```cpp
class Solution {
public:
    int mySqrt(int x) {
        int l = 0, r = x;
        while(l < r){
            int mid = l + 1ll + r  >> 1;
            if(mid <= x/mid) l = mid;
            else r = mid - 1;
        }
        return l;
    }
};
```
