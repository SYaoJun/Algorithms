## 题意

-   [875. 爱吃香蕉的珂珂](https://leetcode-cn.com/problems/koko-eating-bananas/)  M
-   考查知识点：二分

## 代码

-   右模板

-   上取整技巧$\lceil \frac{b}{a} \rceil$ 等价于$\lfloor \frac{b+a-1}{a} \rfloor$

```cpp
class Solution {
public:
    int minEatingSpeed(vector<int>& piles, int h) {
        int l = 1, r = 0;
        for(auto p: piles) r = max(r, p);
        while(l < r){
            int mid = l + r >> 1;
            if(check(mid, piles, h)) r = mid;
            else l = mid + 1;
        }
        return l;
    }
    bool check(int x, vector<int>&piles, int h){
        int sum = 0;
        for(auto p: piles){
            sum += (p+x-1)/x;
        }
        return sum <= h;
    }
};
```
