## 题意

-   求三数之和等于target的数量
-   考查知识点：哈希表

## 代码

-   时间复杂度：$O(N^2)$
-   思路：非常暴力，就是枚举后面两个值，第一个值存在哈希表中，有点像两数之和。

```cpp
class Solution {
public:
    using LL = long long;
    int threeSumMulti(vector<int>& arr, int target) {
        LL res = 0;
        const int MOD = 1e9+7;
        unordered_map<LL, LL> mp;
        int n = arr.size();
        for(int i = 0; i < n; i++){
            for(int j = i+1; j < n; j++){
                int sum = arr[i] + arr[j];
                res = (res + mp[sum]) % MOD;
            }
            mp[target-arr[i]]++;
        }
        return res;
    }
};
```
