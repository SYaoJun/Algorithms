## 题意

-   求一个01数组等于目标值的子数组数量。
-   考查知识点：哈希表+前缀和

## 代码

-   时间复杂度：$O(N)$
-   空间复杂度：$O(N)$

```cpp
class Solution {
public:
    int numSubarraysWithSum(vector<int>& nums, int goal) {
        unordered_map<int, int> hash;
        hash[0] = 1;
        int sum = 0, res = 0;
        for(int i = 0; i < nums.size(); i++){
            sum += nums[i];
            res += hash[sum-goal];
            hash[sum]++;
        }
        return res;
    }
};
```
